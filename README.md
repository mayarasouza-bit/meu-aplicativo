# 📱 Meu Aplicativo

Aplicativo mobile desenvolvido com **React Native, Expo e TypeScript** para praticar desenvolvimento mobile e organização de componentes.

## 🚀 Tecnologias

* React Native
* Expo
* TypeScript

## 📂 Estrutura do projeto

```
meu-aplicativo
 ├── app
 │   └── index.tsx
 ├── components
 │   ├── Header.tsx
 │   ├── Footer.tsx
 │   └── Botao.tsx
 ├── package.json
 ├── tsconfig.json
 └── app.json
```

## ▶️ Como executar

```bash
npm install
npx expo start
```
